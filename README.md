# 🤖 Robot Maze Navigation using Depth First Search (DFS)

## 📌 Project Overview

This project demonstrates how a robot can navigate through a maze using the **Depth First Search (DFS)** algorithm. The program searches for a path from a starting position to a destination while avoiding obstacles (walls) in the maze.

The maze is represented as a 2D grid where:
- **0** → Free path
- **1** → Obstacle/Wall

The algorithm explores one path as deeply as possible before backtracking until the destination is found.

---

## 🚀 Features

- Implements the **Depth First Search (DFS)** algorithm.
- Finds a valid path from the start point to the goal.
- Uses a stack-based traversal.
- Tracks visited nodes to avoid infinite loops.
- Reconstructs the discovered path.
- Supports movement in four directions:
  - Up
  - Down
  - Left
  - Right
